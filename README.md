# cypress-automation-project - E-commerce

End-to-End Automation Framework using Cypress for E-commerce application implementing Page Object Model and reusable test utilities.

##  Project Overview
This project is an End-to-End Automation Framework developed using Cypress for an E-commerce web application.

The framework is designed using Page Object Model (POM) to ensure scalability, reusability, and maintainability of test scripts.

##  Features Covered
- User Registration
- Login Functionality
- Add to Cart
- Checkout Flow
- Product Search
- Filters Validation
- Assertions & Validations

##  Tech Stack
- Cypress
- JavaScript
- Mocha Framework
- Page Object Model (POM)
- Node.js
